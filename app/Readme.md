## 📱 Project Overview
This is a simple Android app built using **Java** and **ConstraintLayout**.  
It collects the user's name, age, and program, then displays an introduction message.

---

## 🧩 Features
- 3 input fields (Name, Age, Program)  
- 1 button (“Introduce Me”)  
- Validation for empty fields and invalid age  
- Output shown dynamically in a TextView  
- Custom theme and colors  

---

## 🧠 Concepts Demonstrated
- UI design with **ConstraintLayout**  
- Event handling with `OnClickListener`  
- Input validation in Java  
- Dynamic TextView updates  
- Custom theming via `themes.xml`  

---

## ⚙️ How to Run
1. Open the project in **Android Studio**.  
2. Click **Run ▶️** to build and launch the app in an emulator or device.  
3. Enter your name, age, and program, then press **“Introduce Me”**.  
4. The output will appear in the TextView below.  
5. Leave any field empty or age = 0 → error message appears.
